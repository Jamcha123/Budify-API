# Budify - Finding Prices 

Budify API is for price discovery for your budget and finding prices for your budget items.

***Budify Input Options***

1. Budget item input e.g Rent, Food, Insurance, Clothes, Electronics, Internet, Energy Or Anything

2. City input e.g London, New York City, Singapore, Oslo, Stockholm Or Etc.

3. Housemate Input e.g 1, 2, 3 or how ever many people you need to pay for


***Budify API***

Curl Option: 

```curl  --header "Authorization: <yourApikey>" "https://pro-budify-2223hqzj3q-uc.a.run.app?q=rent&city=new%20york%20city&housemates=2"```


***Budify Pricing***

Budify API pricing starts at $0.10 per request.

Budify API requires no downpayment or up front cost.

Purely a pay as you go system (you pay only for your usage)


***Budify Use Cases Examples***

1. Price discovery of your items in your shopping list or budget e.g rent, food, insurance, energy, internet or etc. 

2. Developing budget apps e.g using it in the background of your app to find the average and median prices of your items a

3. Managing money better to escape the rat race e.g not being paycheck to paycheck
