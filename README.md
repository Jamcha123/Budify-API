# Budify - Finding Prices 

Budify API is for price discovery for your budget and finding prices for your budget items.

***Budify Input Options***

1. Budget item input e.g Rent, Food, Insurance, Clothes, Electronics, Internet, Energy Or Anything

2. City input e.g London, New York City, Singapore, Oslo, Stockholm Or Etc.

3. Housemate Input e.g 1, 2, 3 or how ever many people you need to pay for


***Budify API***

Curl Option: 

```curl  --header "Authorization: sk-5a9c661e18eb4dc195d8db93" https://pro-budify-2223hqzj3q-uc.a.run.app?q=rent&city=new%20york%20city&housemates=2```
